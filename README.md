# GH-Commands
Este repositorio muestra todos los comandos de gh para conexiones SSH a GitHub

# Lista de comandos

| Comando | Descripción |
| gh auth login | Inicia sesión en tu cuenta de GitHub |
| gh auth status | Verifica si estas conectado correctamente y la cuenta que se usa |
| gh auth logout | Cierra la sesión |
| gh repo create | Crear repositorio |
| gh repo clone <usuario>/<repo> | Clonar un repositorio |
| gh repo view --web | Abre el repositorio actual directamente en el navegador |
| gh repo fork | Crea un fork del repositorio actual |
| gh pr create | Abre un pull request desde terminal |
| gh pr list | Muestra una lista de pr's abiertos |
| gh pr view | Muestra los detalles del pr actual | 
| gh pr checkout <número> | Cambia a la rama del pr para probarlo localmente |
| gh pr merge | Fusiona el pr actual (se puede añadir --squash o --rebase) |
| gh issue create | Crear un nuevo issue |
| gh issue list | Lista los issue del repo actual |
| gh issue close <numero> | Cierra un issue especifico |
| gh issue comment <numero> | Publica un comentario en un issue |
| gh run list | Lista las ejecucciones recientes de los flujos de trabajo | 
| gh run view | Muestra el estado de una ejecuccion especifica |
| gh run watch | Monitorea una ejeccucion en tiempo real |
